# Design Art Architecture Timeline

An interactive Traditional Chinese timeline website for global design history, art history, architecture history, and world context from the Renaissance to the present.

## Contents

- `index.html`: standalone website file.
- Historical images are loaded dynamically from Wikipedia/Wikimedia page summaries.
- If a remote thumbnail is unavailable, the page falls back to an inline SVG placeholder.

## GitHub Pages

Use GitHub Pages with the `main` branch and root folder to publish the site.
